在Hexo中添加Giscus评论系统

前言

在我们使用Hexo博客框架搭建博客的时候，有些时候也许需要读者评论。
但是Hexo是一款纯静态博客框架，评论只能依赖第三方平台进行支撑。
今天我来和大家介绍一个评论系统——Gitscus，并且绑定到博客中。

Gisgus是由由 GitHub Discussions 驱动的评论系统。让访客借助 GitHub 在你的网站上留下评论和反应吧！
该项目受 utterances 强烈启发。

1、创建新仓库

首先在GitHub上创建一个新的仓库，比如我的https://github.com/fxxsj/giscus

2、安装giscus

点击这个链接安装：GitHub Apps - giscus

注意看英文，选择刚建立的仓库，点击install

3、配置仓库discuss部分

首先打开仓库的setting，将Discussions部分打上对号

然后建立一个Announcements的分类,通常是默认存在的。

4、进入giscus配置

链接在这：giscus

填写仓库 我这里是fxxsj/giscus

页面 ↔️ discussion 映射关系 默认第一个即可

Discussion 分类 推荐使用公告（announcements）类型的分类

配置完上述部分后，可以在启动giscus下边复制你主题所需的内容

部署

在主题主题配置文件填写相关的信息，我使用的是stellar主题，_config.stellar.yml
