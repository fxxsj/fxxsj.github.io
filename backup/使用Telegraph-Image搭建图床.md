今天打算试试 tgState：一款以Telegram作为储存的文件外链系统 用来做一个简单的图床，然后浏览文档的时候，发现了一个更加容易搭建的 [Telegraph-Image](https://github.com/cf-pages/Telegraph-Image)。
按使用文档操作非常顺利。
1.首先Fork仓库。
2.打开 Cloudflare Workers 和 Pages，在概述页面中选择创建应用程序，选择连接到 Git 提供程序

3.简单的授权后即可完成部署。
